Regalo de Navidad

Archivos:
- `index.html` — Página principal. Abre este archivo en el navegador.
- `styles.css` — Estilos para la caja, tapa, cinta y mensaje.
- `script.js` — Lógica para el botón `Abrir`.

Uso:
1. Abrir `index.html` en tu navegador (doble clic o arrastrar al navegador).
2. Presiona el botón **Abrir** para que la tapa se levante y aparezca el mensaje.

Personalización rápida:
- Cambia el mensaje en `index.html` (elemento con id `message`).
- Cambia la tipografía modificando el enlace a Google Fonts en `index.html`.

Hecho con ❤️ — si quieres, puedo añadir confeti al abrir o un sonido breve.
